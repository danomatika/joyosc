joyosc icon
===========

Concept generated by the Stable Diffusion AI model from the following prompt:

    app icon, colorful background, happy smiling gamepad, cartoon, cell-shaded

Vectorized by hand in Affinity Designer 2, exported master joyosc_icon.jpg

Create/update all icon file types in `data` with makefile:

    make

Remove generated icon files:

    make clean

Note: this subdirectory is *not* included in the distribution tarball, only the
      icon files in the `data` directory
